Privacy Policy

This Privacy Policy explains what limited information may be collected when you use Champions League Central as a student web project.

1. No account system

This site does not create user accounts or require you to log in. You can browse pages, read content, and submit example forms as part of the project without registering.

2. Forms and example data

The contact form and community forms on this site are for demonstration only. They are not connected to a real database or email system. Any information you type is not stored permanently by the project.

3. Basic analytics and logs

If this project is hosted on a service like GitHub Pages, that service may keep standard web server logs (such as IP address and browser type) for security and traffic statistics. These logs are handled by the hosting provider and not directly by the student building this site.

4. Links to other sites

This site may mention or link to official UEFA or club websites. Those sites have their own privacy policies. Always read their policies before sharing personal information on external sites.

5. Changes to this policy

Because this is a class assignment, the content of this page may be updated as the project evolves. Any significant changes to how data is handled will be reflected here.
